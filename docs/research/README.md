# Research

`docs/research/` 用于存放**当前仍有参考价值的研究型文档**。

这类文档通常具有以下特征：

- 不属于当前权威主文档
- 不属于当前待决实施方案
- 但仍然对上游机制理解、后续增强或故障定位有现实参考价值

规则如下：

1. 当前仍会被引用的研究稿可放在此目录。
2. 若研究结论已经完全被主文档吸收，且后续不再需要直接阅读，可归档到 `docs/archive/analysis/`。
3. 若研究稿转化为当前实施中的待决方案，应移动到 `docs/plans/` 或并入现有主文档。
