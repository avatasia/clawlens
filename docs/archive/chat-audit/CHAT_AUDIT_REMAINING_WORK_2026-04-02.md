- 前端实时联动速度和“当前页面消息上下文感知”仍有提升空间

建议优先级：

- 中

影响：

- 风险较低
- 主要是工程整洁度问题

建议优先级：

## 剩余项 4：旧 run 的历史空明细不自动修复

现状：

- 同步写库修复只影响修复后的新 run
- 修复前已经落成“只有 transcript、没有 llm/tool 明细”的历史 run，不会自动补回

影响：

- 新数据正常
- 旧数据仍会保持旧状态

建议优先级：

- 低

1. logger 自动导入
2. 前端实时层
